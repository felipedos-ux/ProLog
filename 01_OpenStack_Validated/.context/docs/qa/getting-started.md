# How do I set up and run this project?

## Getting Started

### Prerequisites

- Python 3.8+

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd 02_loggpt_small

# Install dependencies
pip install -r requirements.txt
```

### Running

```bash
# See package.json for available scripts
npm run <script-name>
```
